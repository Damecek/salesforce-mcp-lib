# salesforce-mcp-lib

AI-first Salesforce 2GP template repository.

## Scope
- Script-first project operations.
- Scratch org deploy/test validation.
- 2GP packaging and release commands.

Out of scope in template bootstrap: domain business implementation inside force-app.

## Quick Start
```bash
npm run task:prepare
```

## Commands
- `npm run org:fresh`
- `npm run org:deploy`
- `npm run org:test`
- `npm run package:init`
- `npm run release:version`
- `npm run release:promote`
- `npm run release:install`
